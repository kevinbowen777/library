### Installation
 - `git clone https://github.com/kevinbowen777/library.git`
 - `cd library`
 - `mkvirtualenv library` or  `workon library`
 - `poetry install`
 - `poetry run python manage.py runserver`
 - Open browser to http://127.0.0.1:8000
